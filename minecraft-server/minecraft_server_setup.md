## Quickstart Guide

Folge diesen Schritten, um deinen Minecraft-Server mit Docker einzurichten und zu starten:

1. **Repository klonen**:

   ```bash
   git clone <REPOSITORY_URL>
   ```

2. **In das Projektverzeichnis wechseln**:

   ```bash
   cd minecraft-server
   ```

3. **Server starten**:
   Mit Docker Compose den Server im Hintergrund starten:

   ```bash
   docker-compose up -d
   ```

4. **Mit dem Server verbinden**:
   Öffne Minecraft und verbinde dich mit dem Server über deine IP-Adresse und Port 8888.
